# ODI PROJECT FRONTEND-WEB

### Run

```
1. git clone mars_intern_odi_frontend-web
2. cd mars_intern_odi_frontend-web
3. npm install
4. npm run dev
```

### Project Directory

```
├── src
│   ├── components : Components
│   ├── hooks : Custom Hooks
│   ├── store : Redux
│   ├── pages : View
│   ├── services : Api
│   ├── App.jsx
│   └── main.jsx
├── index.html
├── package.json
├── public
```
